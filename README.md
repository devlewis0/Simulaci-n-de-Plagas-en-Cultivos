# Simulación de Plagas en Cultivos

## Descripción

La **Simulación de Plagas en Cultivos** es una herramienta interactiva diseñada para ayudar a los agricultores y agrónomos a visualizar y predecir el impacto de las plagas en sus campos de cultivo. Utilizando un modelo basado en Python y `tkinter` para la interfaz gráfica, esta aplicación permite simular la propagación de plagas y su efecto sobre los cultivos a lo largo del tiempo.

## Funcionalidades

- **Interfaz Gráfica Intuitiva**: Utiliza `tkinter` para proporcionar una interfaz amigable donde los usuarios pueden ingresar datos y ver los resultados.
- **Simulación de Propagación de Plagas**: Modelo que simula la reproducción y mortalidad de plagas a lo largo del tiempo.
- **Visualización de Resultados**: Gráficos generados con `matplotlib` para mostrar el estado inicial del campo, la distribución de plagas después de la simulación y la evolución de la población de plagas.
